A Ruby wrapper for VoiceIt's API 3.0 featuring Voice + Face Verification and Identification.

## Installation

```bash
git clone https://github.com/voiceittech/voiceit3-ruby.git
```

Then require in your project:
```ruby
require_relative "voiceit3-ruby/voiceit3"
```

## Getting Started

Sign up at [voiceit.io/pricing](https://voiceit.io/pricing) to get your API Key and Token, then log in to the [Dashboard](https://dashboard.voiceit.io) to manage your account.
